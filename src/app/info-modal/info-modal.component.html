Info & Credits

This tool depends on data from others. Some of the most frequent sources of that data are:

  • NOT A HOAX! NOT A DREAM!

    We rely on the reviews on this blog to know which order the comics appear in within collections.

  • Comics Back Issues

    We rely on this to track and order crossovers.

  • Marvel

    We rely on the Marvel API to display the link to the online version of the comics.
